
## Recipes API
This is a basic RESTful API that serves as a proxy to an existing weather API
It implements generic tests with mocks.

### Requirements:

- Git.
- Composer.
- PHP 7.3+.
- Guzzle

### Running the app:

The app can be run using the following commands:

```bash
$ cd [api-name]
$ composer test
$ composer start
```
